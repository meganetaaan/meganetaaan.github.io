子持ち家庭の「ガジェットどこに置く問題」

  • スマートスピーカー、スマートリモコン、プロジェクターをリビングに置いている
  • これらはすべて「声が届く場所」「電波が届く場所」「壁からの投影距離」などの制約がある
  • 分かりやすい場所に置くと子供（3歳）が触りたがる
  • これを静止するのは（リビングの主たる運用者である）妻の負担となる

解決策としての「天吊り」

@r7kamuraの天吊りプロジェクターの記事に着想

天吊りのメリット

  • 子供が触れなくなることで設備保全に繋がる
  • 「プロジェクターのレンズを直接覗きこむ」など危険な行動を抑制できる
  • 部屋がすっきりする

天吊りのデメリット

  • 自分もデバイスを取り外しづらくなる（基本設置しっぱなしなのでそこまで不便は感じない）
  • 落下の危険がある（安全措置を必ず講じること）
